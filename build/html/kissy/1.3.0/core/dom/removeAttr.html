方法¶

dom.removeAttr()¶
void removeAttr ( selector, name )
移除符合选择器的所有元素的指定属性.
参数:
  • selector (string|HTMLCollection|Array<HTMLElement>) – 字符串格式参见 KISSY selector
  • name (string) – 属性名称 , 必须是以“data-”开头的字符串

例如

<input type="radio" data-checked="checked">

var DOM = KISSY.DOM;
        // 将所有复选框的 checked 属性移除
        DOM.removeAttr("input","data-checked");
        DOM.hasAttr("input","data-checked");  //false
