组件¶

node

方法¶

node.contains()¶
Boolean contains (container , contained)
判断某一容器(container)是否包含另一(contained)节点.
参数:
  • container (string|HTMLElement|ArrayList<HTMLElement>|HTMLDcoument|TextNode) –

    容器节点.

    • 字符串格式参见 KISSY selector 获取匹配的第一个元素.
    • ArrayList<HTMLElement> : 取列表第一个元素
  • contained (string|HTMLElement|ArrayList<HTMLElement>|HTMLDcoument|TextNode) –

    检测节点.

    • 字符串格式参见 KISSY selector 获取匹配的第一个元素.
    • ArrayList<HTMLElement> : 取列表第一个元素
返回:

container 是否包含 contained 节点.
返回类型:

Boolean

注解

两个元素如果相等, 则返回 false
