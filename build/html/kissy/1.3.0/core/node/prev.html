dom.prev()¶
HTMLElement prev (selector [ , filter ])
获取符合选择器的第一个元素的上一个同级节点.
参数:
  • selector (string|HTMLCollection|Array<HTMLElement>) – 字符串格式参见 KISSY selector
  • filter (string|function) – 过滤条件, 格式参见 dom.filter 的相应参数
返回:

符合选择器的第一个元素的上一个同级节点.
返回类型:

HTMLElement
